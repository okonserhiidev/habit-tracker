# Стек технологий — Habit Tracker (финальный)

## Frontend

| Технология | Назначение |
|-----------|-----------|
| React Native (Expo) | iOS-приложение |
| React (React Native Web) | Веб-версия |
| TypeScript | Язык |
| Expo Router | Навигация и роутинг |
| Zustand | Управление состоянием (клиентское) |
| TanStack Query | Управление состоянием (серверное, кэш API) |
| Axios | HTTP-клиент |

## Backend

| Технология | Назначение |
|-----------|-----------|
| Java | Язык |
| Spring Boot | Фреймворк |
| Spring Data JPA + Hibernate | ORM |
| Spring Security + JWT | Авторизация |
| Flyway | Миграции базы данных |

## База данных

| Технология | Назначение |
|-----------|-----------|
| PostgreSQL | Основная БД |
