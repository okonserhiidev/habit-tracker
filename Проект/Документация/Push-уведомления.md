# Push-уведомления в Habit Tracker

## Что такое push-уведомления

Push-уведомления — это сообщения, которые приложение отправляет на телефон пользователя, **даже когда приложение закрыто**. Они появляются на экране блокировки, в шторке уведомлений и могут содержать текст, звук и вибрацию.

### Пример для Habit Tracker:
- 🔔 "Не забудь выпить воду!" — в 9:00 утра
- 🔔 "Ты на 5-дневном streak'е по чтению! Не останавливайся" — вечером
- 🔔 "Ты ещё не отметил 3 привычки сегодня" — напоминание

---

## Как это работает

```
Телефон пользователя
        ↑
        | push-уведомление
        |
   Apple APNs (Apple Push Notification service)
        ↑
        | отправка уведомления
        |
   Firebase Cloud Messaging (FCM)
        ↑
        | запрос на отправку
        |
   Наш Backend (Spring Boot)
        ↑
        | триггер: расписание / событие
        |
   Логика приложения
```

### Шаг за шагом:
1. Пользователь открывает приложение → оно получает **device token** (уникальный ID устройства)
2. Приложение отправляет этот token на наш backend
3. Backend сохраняет token в БД
4. Когда нужно отправить уведомление → backend отправляет запрос в **Firebase Cloud Messaging (FCM)**
5. FCM передаёт уведомление через **Apple APNs** на iPhone пользователя

---

## Два типа уведомлений

### 1. Локальные уведомления
- Срабатывают **без сервера**, по расписанию на самом телефоне
- Например: "Каждый день в 9:00 — напомни про зарядку"
- Работают оффлайн
- Для Habit Tracker — **основной тип** (напоминания по расписанию)

### 2. Серверные push-уведомления
- Отправляются **с сервера** через FCM
- Например: "Твой друг тоже начал трекать привычки!" или "Новая фича!"
- Нужен интернет
- Для Habit Tracker — для социальных фич, маркетинга, streak-мотивации

---

## Что используем

| Тип | Технология | Когда |
|-----|-----------|-------|
| Локальные | expo-notifications | Напоминания по расписанию привычек |
| Серверные | Firebase Cloud Messaging (FCM) + Spring Boot | Streak-мотивация, социальные фичи |

---

## Зачем это Habit Tracker

Push-уведомления — **критическая фича** для трекера привычек:

1. **Напоминания** — пользователь забудет без них, retention упадёт
2. **Мотивация** — "7 дней подряд! Продолжай!" повышает вовлечённость
3. **Возврат пользователей** — если пользователь не заходил 3 дня, push напомнит
4. **Кастомизация** — пользователь сам выбирает время и текст напоминания для каждой привычки

---

## Стоимость

- **Локальные уведомления** — бесплатно (работают на устройстве)
- **Firebase Cloud Messaging** — бесплатно (без лимитов на количество)
- **Apple Developer Account** — $99/год (нужен в любом случае для публикации в App Store)
