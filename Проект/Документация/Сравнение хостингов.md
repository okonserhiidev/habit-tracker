# Сравнение хостингов для Habit Tracker

## Что нужно хостить

| Компонент | Что это |
|-----------|--------|
| Backend (Spring Boot) | Java-приложение, REST API, ~512 MB–1 GB RAM |
| PostgreSQL | База данных |
| Web (React) | Статические файлы (HTML/JS/CSS) |

---

## Вариант 1 — Railway

| Параметр | Значение |
|----------|---------|
| Backend | Деплой из GitHub, автоматический |
| PostgreSQL | Встроенный managed PostgreSQL |
| Цена | $5/мес (Hobby) — 8 GB RAM, 8 vCPU shared |
| Free tier | Нет (отменили) |
| Масштабирование | Вертикальное, до нескольких инстансов |
| SSL | Автоматический |
| Регионы | US, EU |

**Плюсы:** всё в одном месте (backend + БД), деплой в один клик из GitHub, логи и мониторинг из коробки.

**Минусы:** нет бесплатного плана, при росте дороже чем VPS.

---

## Вариант 2 — Render

| Параметр | Значение |
|----------|---------|
| Backend | Web Service, деплой из GitHub |
| PostgreSQL | Managed PostgreSQL (free — 90 дней, потом $7/мес) |
| Цена backend | Free (750 ч/мес, засыпает) или $7/мес (Starter, не засыпает) |
| SSL | Автоматический |
| Регионы | US, EU (Frankfurt) |

**Плюсы:** есть бесплатный план для старта, простой интерфейс, managed PostgreSQL.

**Минусы:** бесплатный инстанс засыпает через 15 мин без запросов (холодный старт ~30 сек для Java), бесплатная БД удаляется через 90 дней.

---

## Вариант 3 — VPS (Hetzner)

| Параметр | Значение |
|----------|---------|
| Сервер | CX22: 2 vCPU, 4 GB RAM — €4.5/мес |
| PostgreSQL | Ставишь сам на тот же сервер |
| Цена | €4.5–8/мес (всё включено) |
| SSL | Let's Encrypt (настраиваешь сам или через Caddy) |
| Регионы | EU (Германия, Финляндия), US |

**Плюсы:** дешевле всех, полный контроль, мощное железо за свои деньги, всё на одном сервере.

**Минусы:** нужно самому настраивать (Docker, Nginx/Caddy, SSL, бэкапы, обновления), нет автодеплоя из коробки (но решается через GitHub Actions).

---

## Вариант 4 — DigitalOcean App Platform

| Параметр | Значение |
|----------|---------|
| Backend | App Platform, деплой из GitHub |
| PostgreSQL | Managed Database — от $15/мес |
| Цена backend | $5/мес (Basic) |
| SSL | Автоматический |

**Плюсы:** надёжная платформа, хорошая документация, managed БД.

**Минусы:** managed PostgreSQL дорогой ($15/мес), общая цена выше.

---

## Хостинг для Web (React)

Статический сайт — хостится отдельно и бесплатно:

| Платформа | Цена | Особенности |
|-----------|------|------------|
| **Vercel** | Бесплатно (Hobby) | Автодеплой из GitHub, CDN, идеально для React |
| **Netlify** | Бесплатно (Starter) | То же самое, чуть другой интерфейс |
| **GitHub Pages** | Бесплатно | Самый простой вариант |

→ Рекомендация: **Vercel** — лучшая интеграция с React-проектами.

## Рекомендация

### Для старта (MVP):
**Render (backend) + Vercel (web)** — бесплатный старт, минимум настройки, быстрый деплой.

### Для продакшена:
**Hetzner VPS + Vercel (web)** — дешевле всех ($4.5/мес за всё), полный контроль, Java не засыпает. Настроить Docker + GitHub Actions один раз — дальше автодеплой.

### Оптимальный путь:
1. Начать на **Render** (бесплатно, быстро)
2. Когда проект вырастет — переехать на **Hetzner VPS** (дёшево и надёжно)
