# Техническое задание — Habit Tracker

## Цель проекта

Создать приложение для отслеживания привычек с детальной кастомизацией и глубокой аналитикой, доступное на iOS и в браузере, с синхронизацией данных между платформами. Дизайн и UX основаны на принципах книги "Атомные привычки" Джеймса Клира.

## Целевая аудитория

Люди, которые хотят формировать полезные привычки и отслеживать свой прогресс: студенты, специалисты, все кто работает над самодисциплиной.

## Функциональные требования

### F1. Регистрация и авторизация
- [ ] Вход через Google (OAuth2) — основной способ
- [ ] Вход через email + пароль — альтернативный способ
- [ ] Вход / выход
- [ ] JWT-токен для сессий
- [ ] Восстановление пароля (для email-регистрации)

### F2. Управление привычками
- [ ] Кнопка "Добавить привычку" на главном экране
- [ ] Название привычки
- [ ] Описание привычки
- [ ] Выбор цвета через RGB-палитру (полный спектр, не предустановленные цвета)
- [ ] Выбор иконки из набора
- [ ] Редактирование привычки
- [ ] Удаление привычки
- [ ] Архивирование привычки (скрыть, но сохранить историю)
- [ ] Шаблоны привычек по категориям (здоровье, продуктивность, спорт, учёба, отношения)
- [ ] Группировка привычек по категориям

### F3. Отметка выполнения
- [ ] Галочка (чекбокс) на главном экране — **один тап** для отметки (без подтверждений)
- [ ] Микроанимация при отметке (pulse-эффект в цвете привычки, вибрация)
- [ ] Отменить отметку (если нажал случайно)
- [ ] Отметка за прошлые дни (backfill)
- [ ] Прогресс-бар дня: "5 из 7 привычек выполнено"
- [ ] Анимация "Perfect Day!" когда выполнены все привычки за день

### F4. Детальный просмотр привычки
При нажатии на привычку открывается детальный экран:

#### Календарь
- [ ] Месячный календарь с отмеченными днями выполнения
- [ ] Цвет отметок = цвет привычки (выбранный через RGB)
- [ ] Переключение между месяцами
- [ ] Возможность отметить/снять выполнение прямо из календаря

#### Streak
- [ ] Текущий streak (дней подряд) с fire-иконкой 🔥
- [ ] Лучший streak за всё время
- [ ] Визуальное отображение streak (fire-анимация, рост при каждой отметке)
- [ ] Предупреждение при угрозе потери streak'а: "Ты на 14-дневном streak'е!"

#### Heatmap
- [ ] Годовой heatmap (как на GitHub)
- [ ] Интенсивность цвета = стабильность выполнения
- [ ] Цвет heatmap = цвет привычки

### F5. Аналитика (детальная)
Отдельный раздел с глубокой аналитикой по каждой привычке и по всем вместе:

#### По отдельной привычке
- [ ] Процент выполнения: за неделю / месяц / квартал / год / всё время
- [ ] График выполнения по дням (line chart) — тренд за период
- [ ] Лучшие и худшие дни недели (в какой день чаще выполняешь)
- [ ] Лучшие и худшие месяцы
- [ ] Средний streak
- [ ] Общее количество выполненных дней
- [ ] Пропущенные дни подряд (антистрик)
- [ ] Сравнение текущего месяца с предыдущим (прогресс/регресс)
- [ ] Тренд улучшения: "На этой неделе ты на 3% лучше, чем на прошлой"

#### Общий dashboard
- [ ] Общий процент выполнения всех привычек за период
- [ ] Топ привычек по стабильности
- [ ] Привычки, которые "проседают" (требуют внимания)
- [ ] Общий график активности по дням недели (radar chart)
- [ ] Тренд общей дисциплины за месяц/квартал (line chart)
- [ ] Количество активных привычек
- [ ] "Идеальные дни" — дни когда выполнены все привычки
- [ ] Серия идеальных дней (perfect streak)
- [ ] Рейтинг дисциплины с трендом: "78% → 81% (+3%)"

#### Визуализации (типы графиков)
- [ ] Line chart — тренды по времени
- [ ] Bar chart — сравнение по дням недели / месяцам
- [ ] Radar chart — профиль дисциплины по дням недели
- [ ] Heatmap — годовой обзор
- [ ] Pie / donut chart — распределение по категориям
- [ ] Progress rings — процент выполнения за период

### F6. Уведомления и мотивация
- [ ] Локальные напоминания по расписанию для каждой привычки
- [ ] Настройка времени напоминания при создании привычки
- [ ] Включение / отключение уведомлений для каждой привычки
- [ ] Утренний push: план дня ("Сегодня у тебя 5 привычек")
- [ ] Вечерний push: "Ты ещё не отметил 2 привычки"
- [ ] Milestone-уведомления: 7 дней, 21 день, 30 дней, 66 дней, 100 дней
- [ ] Мотивирующие сообщения в первые 21 день (критический период формирования привычки)
- [ ] (позже) Серверные push: мотивация по streak'ам, возврат неактивных пользователей

### F7. Синхронизация
- [ ] Данные синхронизируются между iOS и Web через REST API
- [ ] Привязка данных к аккаунту пользователя
- [ ] (позже) Оффлайн-режим с синхронизацией при восстановлении сети

### F8. Персонализация интерфейса
- [ ] Тёмная / светлая тема
- [ ] Порядок привычек (drag & drop)
- [ ] Настройка главного экрана (какие привычки показывать)

### F9. Фичи из "Атомных привычек"

#### Правило двух минут
- [ ] При создании привычки — подсказка "Начни с малого"
- [ ] Возможность задать "мини-версию" привычки для трудных дней
- [ ] Пример: "Бегать 30 минут" → предложение: "Надеть кроссовки и выйти на улицу"

#### Наложение привычек (Habit Stacking)
- [ ] Связывание привычек в цепочки (утренний ритуал, вечерний ритуал)
- [ ] Визуальное отображение цепочки
- [ ] Уведомление о следующей привычке в цепочке после отметки предыдущей

#### Идентичность
- [ ] Поле "Кем я хочу стать?" при создании привычки (необязательное)
- [ ] Пример: привычка "Читать" → идентичность "Я — читающий человек"
- [ ] Мотивирующие фразы на dashboard, привязанные к идентичности

#### Freeze Day
- [ ] Возможность "заморозить" streak 1 раз в неделю
- [ ] Freeze day не считается пропуском, streak сохраняется
- [ ] Ограниченное количество freeze days (1-2 в неделю)

#### Долина разочарования
- [ ] Мотивирующие сообщения в первые недели: "Ты в Долине разочарования — это нормально!"
- [ ] Визуализация "Плато скрытого потенциала" — показать, что прогресс копится

#### Онбординг
- [ ] Приветственный тур при первом входе
- [ ] "Давай составим список твоих привычек" — помощь в создании первых привычек
- [ ] Шаблоны по категориям для быстрого старта

### F10. Геймификация и награды
- [ ] Достижения за milestone'ы (7, 21, 30, 66, 100, 365 дней)
- [ ] Разблокировка тем/иконок за достижения
- [ ] Еженедельный "отчёт успеха" — подведение итогов с акцентом на позитив
- [ ] Звуковые эффекты при отметке (с возможностью отключения)

## Нефункциональные требования

### Производительность
- Время ответа API: < 500 мс
- Время запуска приложения: < 3 сек
- Плавные анимации: 60 fps
- Графики рендерятся без задержки при большом объёме данных
- Главный экран открывается сразу на "Сегодня" — без лишних экранов

### Безопасность
- Хранение паролей: bcrypt
- Все запросы через HTTPS
- JWT с refresh token
- Валидация входных данных на backend

### Масштабируемость
- Stateless API (JWT)
- Архитектура позволяет горизонтальное масштабирование

### Доступность
- iOS 15+
- Современные браузеры (Chrome, Safari, Firefox)

### UX-принципы (из "Атомных привычек")
- Отметка привычки — один тап, без модальных окон
- Создание привычки — максимум 3 шага (название → цвет → расписание)
- Каждое действие пользователя = визуальная обратная связь
- Фокус на прогрессе, а не на провалах

## Экраны приложения

1. **Онбординг** — приветствие, создание первых привычек из шаблонов
2. **Вход / Регистрация** — Google OAuth + email/пароль
3. **Главный экран** — список привычек с чекбоксами на сегодня + прогресс-бар дня
4. **Создание / редактирование привычки** — название, описание, RGB-палитра, иконка, мини-версия, идентичность
5. **Детальный экран привычки** — календарь, streak, heatmap
6. **Аналитика привычки** — все графики по одной привычке
7. **Общий dashboard** — аналитика по всем привычкам, рейтинг дисциплины
8. **Цепочки привычек** — утренний/вечерний ритуал (Habit Stacking)
9. **Достижения** — milestone'ы, разблокированные награды
10. **Настройки** — профиль, тема, уведомления, звуки
